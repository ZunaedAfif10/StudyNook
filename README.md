## 🚀 Project Overview

StudyNook was developed to create a more structured and accessible learning environment for students. The platform eliminates the hassle of scattered notes and disconnected resources by offering an organized ecosystem for educational content management.

With a clean interface and responsive architecture, users can seamlessly browse, upload, manage, and access study materials from any device.

### 🎯 Key Objectives

- Simplify academic resource management
- Improve collaborative learning experiences
- Provide a responsive and intuitive UI
- Ensure secure authentication and data handling
- Deliver scalable full-stack architecture

## ⚙️ Installation & Local Development Setup

Follow the steps below to run the project locally on your machine.

</div>

---

# 📥 Clone the Repositories

### Client Side

```bash
git clone https://github.com/ZunaedAfif10/StudyNook.git
```

### Server Side

```bash
git clone https://github.com/ZunaedAfif10/StudyNook-server.git
```

## 📦 Install Dependencies

### Client Setup

```bash
cd StudyNook
npm install
```

## 🔑 Environment Variables

### Create a ```.env``` file in both client and server directories.

## ▶️ Run the Project

### Start Client

```bash
npm run dev
```

### Start Server

```bash
nodemon index.js
```
